# Vendoraft

Inventory + POS Lite for Small Businesses

A full-stack business system with product management, stock tracking,
sales recording, reporting, and a dashboard. Built with Next.js,
Node.js, Express, PostgreSQL (Neon), JWT authentication, and deployed
on Vercel + Render.
